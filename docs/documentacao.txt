Instalação e configuração do ambiente na VM
Passo 1: acessar a pasta do projeto
- cd /home/univates/task2-fabri

Passo 2: instalar as dependências da aplicação
- npm install --omit=dev

Passo 3: criar o usuário do PostgreSQL para a aplicação
- sudo -u postgres psql
- CREATE ROLE financeiro_app WITH LOGIN PASSWORD 'financeiro123';
- \q

Passo 4: criar o banco de dados
- sudo -u postgres createdb -O financeiro_app financeiro_db

Passo 5: executar o script de criação das tabelas
- PGPASSWORD='financeiro123' psql -h 127.0.0.1 -U financeiro_app -d financeiro_db -f sql/schema.sql

Passo 6: executar o script de carga inicial
- PGPASSWORD='financeiro123' psql -h 127.0.0.1 -U financeiro_app -d financeiro_db -f sql/seed.sql

Passo 7: criar o arquivo .env da aplicação
- PORT=3000
- DB_HOST=127.0.0.1
- DB_PORT=5432
- DB_NAME=financeiro_db
- DB_USER=financeiro_app
- DB_PASSWORD=financeiro123

Passo 8: liberar a porta da aplicação no firewall
- sudo ufw allow 3000/tcp

Passo 9: configurar o serviço da aplicação
- sudo systemctl daemon-reload
- sudo systemctl enable --now task2-fabri.service
- sudo systemctl restart task2-fabri.service

Passo 10: verificar se o serviço está ativo
- sudo systemctl status task2-fabri.service

Comandos úteis do serviço
